Curso de Git e GitHub

Conceitos principais

Commit: Envio de alterações feitas dentro de um código para um repositório local. (Git)

Push: Envio de alterações do repositório local para um repositório remoto. (GitHub)

Origin: Mandar seus versionamentos (commits) do repositório local para o repositório remoto.

Branch

O conceito de branch vem de "ramo", "ramificação", sendo entendida como galhos de uma árvore, a branch master seria nosso ramo principal, o qual será alocado nosso proejto final, o ideal é não fazer commits dentro da main/master, e sim em outros ramos, como exemplo branch de conteudo e de design.
